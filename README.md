# Landing Page Project

A professional, high-performance landing page built with Next.js, featuring dynamic content management via Hygraph Headless CMS. This project utilizes a component-driven architecture to ensure scalability and easy content updates.


- **DEMO:** [https://byteex-nine.vercel.app/]
- **GITHUB:** [https://github.com/MarinaKramarchuk/byteex]

## 🛠 Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **CMS:** Hygraph (Headless GraphQL CMS)
- **Styling:** Tailwind CSS
- **Data Fetching:** GraphQL / Fetch API
